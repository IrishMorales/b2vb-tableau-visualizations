# B2VB Tableau Visualizations
Data visualizations to learn the basics of Tableau. Created as part of [Back 2 Viz Basics](https://www.thetableaustudentguide.com/vizbasics), a set of bi-weekly challenges to create different kinds of data visualizations and emphasize use of effective data storytelling.
